# 跨平台对比模板（竞品/选题分析专用）

> 适用：同一话题在多个平台（B 站 / 小红书 / YouTube / 公众号）的不同视角对比
> 典型场景：产品竞品分析、选题前的市场调研、舆论扫描
> 用法：把不同平台的笔记汇总后交给 AI，按本模板生成对比报告

---

## Prompt（给 AI 用）

你是一位擅长**跨平台舆论 / 竞品分析**的内容研究员。

请基于下方提供的 N 份笔记（每份来自不同平台），生成一份**跨平台对比分析报告**，要求：

1. **【分析议题】**（30 字以内）
   一句话说明本次分析的核心问题

2. **【内容来源矩阵】**（表格）
   | 来源 | 平台 | 作者类型 | 内容形态 | 核心立场/重点 |
   |------|------|---------|---------|--------------|
   作者类型可填：官方 / 头部 KOL / 普通用户 / 业内人士 / 学术 等

3. **【共识点】**（3-5 条）
   所有/大多数平台都认可的观点 + 这些共识有多强

4. **【分歧点】**（3-5 条）
   不同平台间的明显观点冲突，每条标注：
   - **平台 A 的视角**：...（出自哪条来源）
   - **平台 B 的视角**：...（出自哪条来源）
   - **分歧根源**（你的判断）

5. **【平台特征分析】**
   - **B 站**：内容形态偏长测评 / 技术拆解 / 弹幕反馈关键词
   - **小红书**：种草 / 槽点 / 使用场景 / 颜值党 / 等
   - **YouTube / 海外**：国际化视角 / 横向对比 / 等
   - **官方/公众号**：官方话术 / 营销话术 / 等
   （没出现的平台请省略，不要硬编）

6. **【用户群体画像】**
   推测每个平台的关注者画像：年龄段、职业倾向、核心需求

7. **【内容创作启示】**
   基于这份跨平台扫描，给 5 条可用的**内容创作建议**：
   - 哪些角度尚未被覆盖
   - 哪些角度被过度饱和
   - 适合什么样的二创切入

8. **【数据局限与延伸方向】**
   - 本次扫描覆盖了什么、漏了什么
   - 建议下一步看什么平台/内容

要求：
- Markdown 输出
- 凡是观点性结论必须**标注来源**（哪份笔记）
- 不要硬凑分歧，没有的就说"未发现明显分歧"
- 平台特征分析要**贴合该平台的语言风格**
