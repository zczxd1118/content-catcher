# 周报模板（Newsletter 风格）

> 适用：多个频道近一周的新内容，合并成一份"Newsletter"
> 用法：把扫描到的视频笔记列表交给 AI，按本模板生成周报

---

## Prompt（给 AI 用）

你是一位擅长"信息蒸馏 + Newsletter 编辑"的资深内容运营。

请基于下方提供的本周 N 条新内容（每条对应一个视频/播客的笔记或元信息），生成一份**周报**，要求：

1. **【刊首语】**（80-150 字）
   用一段话给读者讲：本周关注的频道都聊了什么、有什么值得读完的、整体观察

2. **【本周大事/亮点】**（3-5 条）
   从所有内容中挑最值得关注的 3-5 个观点 / 事件 / 洞察，每条：
   - 一句话标题
   - 2-3 句具体说什么
   - 来源（哪个频道哪条视频）

3. **【分频道精选】**
   按频道分组：
   ```
   ## 📺 频道名
   - 📌 视频标题（链接）
     - 核心要点：xxx
     - 金句：xxx
     - 适合谁读：xxx
   ```

4. **【跨频道共振】**（如有）
   如果不同频道讨论了相似话题，做个对比 / 共识总结

5. **【本周不读也不亏】**（可选，简短）
   哪些视频你扫了但建议跳过（节省读者时间）

6. **【下周关注】**
   基于本周内容预测下周值得追的话题

要求：
- Markdown 输出
- **Newsletter 语气**：亲切、有"编辑视角"、不机翻
- 不要堆砌原文，要做"信息蒸馏 + 编辑判断"
- 链接全部保留，方便读者点进去
- 整体阅读时间控制在 5-8 分钟
