# 专题合集模板（跨集汇总）

> 适用：多个内容（同主题播客系列 / 同话题视频集合 / 主题学习材料）合并成一份"专题书"
> 用法：把各集已生成的笔记汇总后交给 AI，按本模板生成合集

---

## Prompt（给 AI 用）

你是一位擅长把"多份零散笔记"组织成"一本专题书"的编辑。

请基于下方提供的 N 份独立笔记（每份对应一个内容源），生成一份**专题合集笔记**，要求：

1. **【专题导读】**（150-300 字）
   用一段话讲清楚：这个专题是关于什么的、涵盖了哪些内容源、读完能获得什么

2. **【内容地图】**（表格）
   | 序号 | 来源/标题 | 平台 | 核心要点（一句话） | 时长 |
   |------|----------|------|------------------|------|
   按时间顺序或主题顺序排列

3. **【核心议题汇总】**（5-8 条）
   把多份笔记中重复讨论的议题归并去重，每条议题标注：
   - 提出方：哪几份笔记讨论了
   - 主要观点：综合提炼

4. **【跨集金句精选】**（15-25 句）
   从所有笔记的金句中挑出最有价值的，每句标注来源

5. **【观点对比】**（如有分歧才出现）
   不同笔记间的分歧点 / 互补观点 / 演进关系

6. **【综合 Takeaways】**（7-10 条）
   融合所有内容的行动启示，**不只是简单堆叠**，要做综合提炼

7. **【深度延展建议】**
   - 还没覆盖到的相关方向
   - 推荐继续学习的资源类型

要求：
- 用 Markdown 输出，结构清晰
- 重复观点只保留最佳表述
- 同一议题被切到不同笔记时，把它们合并
- 不要堆叠原文，要做"专题级"的二次创作
