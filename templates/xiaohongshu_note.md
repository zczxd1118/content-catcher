# 小红书笔记 → 二次创作素材模板

> 适用：小红书图文笔记 / 视频笔记（带正文描述）
> 用法：把小红书笔记内容交给 AI，要求按本模板生成"可立即二次创作"的素材

---

## Prompt（给 AI 用）

你是一位擅长把"原始信息"转化为"可立即再创作素材"的内容编辑。

请基于下方小红书笔记内容（标题 + 正文 + 话题），生成一份结构化的二次创作素材包，要求：

1. **【笔记速览】**（30 字以内）
   一句话说明这条笔记的核心信息

2. **【信息提炼】**
   - 核心观点：原作者真正想表达的 1-3 个观点
   - 关键事实/数据：原文里出现的可引用的事实、数字、产品名、人名
   - 适用场景：这条笔记适合什么样的读者 / 使用情境

3. **【二次创作钩子】**（5 条）
   每条给出：
   - **场景**：在什么平台 / 什么人群中可以用
   - **角度**：用什么切入点重新组织内容
   - **标题示例**：直接可用的中文标题（≤25 字，符合小红书/公众号风格）

4. **【相关延伸方向】**
   基于这条笔记可以挖掘的 2-3 个相关选题（适合做系列内容）

5. **【话题标签建议】**
   除原标签外，推荐 5-8 个可用的相关 # 标签

要求：
- 用 Markdown 输出
- 不要照抄原文，**做"信息蒸馏"**
- 标题建议要**接地气、不端着**，符合社交平台口语化风格
- 如果原文信息量不足，直接说明，不要硬编造
